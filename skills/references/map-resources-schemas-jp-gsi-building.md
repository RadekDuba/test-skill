# JP GSI Building

Source: https://docs.maptiler.com/schema/jp-gsi-building/

Tileset containing vector polygons of building footprints.

GSI Building is a tileset that contains vector polygons of building footprints provided by the Geospatial Information Authority of Japan.

## Layers

### gsi_building

Polygons of building footprints.

### Fields

#### type

Building type

Possible values:

- `不明` - unknown
- `堅ろう建物` - robust building
- `堅ろう無壁舎` - robust wall house
- `普通建物` - ordinary building
- `普通無壁舎` - ordinary wallless house
