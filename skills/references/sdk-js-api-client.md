# Source: https://docs.maptiler.com/sdk-js/api/api-client/

Our map SDK is not only about maps! The SDK also wraps API calls to [MapTiler API services](https://docs.maptiler.com/cloud/api) in a series of handy functions.
  The SDK uses the [API Client JS library](/client-js/) under the hood and exposes its functions so that
    they can be used directly from the SDK.
