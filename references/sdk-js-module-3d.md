With the [MapTiler SDK](/sdk-js/) 3D JS module, you can add 3D objects to your basemap with plenty of customizations from **glTF/glb files**. These can be meshes, groups of meshes, point clouds, or a mix of all these.

## Get started

Check out the [Add a 3D model on a map example](/sdk-js/examples/3d-js-get-started/).

## API reference

The [reference](/sdk-js/modules/3d/api/) documents every object and method available. Each section describes classes or objects as well as their **properties, parameters, methods**, and associated **events**. Many sections also include inline code examples and related resources.

The **main class** in the MapTiler 3D JS module is the [**Layer3D class**](/sdk-js/modules/3d/api/classes/Layer3D/)
